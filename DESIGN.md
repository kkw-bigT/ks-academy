## 3. 레이아웃 원칙

- 여백 충분히 — 답답하지 않게
- 모서리: 카드 12–20px / 박스 4–8px
- 그림자: `0 4px 16px rgba(0,0,0,0.08)` (웹앱)

## 4. 산출물별 스펙

### 4-1. 학원 교재기반 학습지
포맷: A4 PDF (LaTeX) | 색상: #2E6E8E + #D89B2E
3종 세트: 학생용(빈칸) / 학생마무리(VOCA+Summary) / 교사용(+정답)

### 4-2. 웹앱
포맷: 단일 HTML (서버리스, localStorage) | 색상: #f59e0b + #1e3a5f + #fafaf7
기술: CSS 변수 필수, 모바일 우선, Font Awesome 허용

### 4-3. SNS 카드뉴스 (인스타그램)
포맷: HTML 500×500px 카드 → 스크린샷 | 색상: #C8A84B + #1C1B18 + #FAF7F0
구조: [표지 Dark] → [본문 White/Cream] → [CTA Mustard]

### 4-4. 시험지
포맷: A4 PDF | 색상: 흑백 기반 + Brand Red 최소화
원칙: 가독성 최우선, 11pt 이상, 1–2단

### 4-5. 발표자료
포맷: PPTX 또는 HTML | 색상: #1e3a5f (메인) + #f59e0b (포인트) + White
구조: [표지 Navy] → [목차] → [본문 White+Amber선] → [마무리]
