# SKILL: 입시 일일 브리핑

## 목적
2026년 입시 관련 핫이슈를 자동 수집·요약하여 카카오톡으로 전송

## 트리거
- 매일 오전 9시 (스케줄러 자동 실행)
- 수동: "오늘 입시 이슈 알려줘"

## 절차

### 1. 검색 (NaverSearch MCP, sort: date)
- 2026 수능 / 2026 대입 변경 / 입시 이슈 / 수시 정시 2026

### 2. 요약 형식 (200자 이내)
```
📚 입시브리핑 [MM/DD]
🔥[이슈1 핵심 1줄]
🔥[이슈2 핵심 1줄]
🔥[이슈3 핵심 1줄]
💡[오늘의 핵심 액션]
```

### 3. 전송
KakaotalkChat-MemoChat으로 나에게 보내기

## 알려진 한계
- 스레드/인스타그램 직접 검색 불가 → 네이버로 대체
- 카카오톡 200자 제한

## 개선 이력
- 2026-07-12: 스킬 최초 생성
