# SKILL: 세션 종료 후 자기개선 반성

## 목적
매 세션 후 에이전트가 스스로 평가하고 MEMORY.md / SKILLS/ 업데이트

## 트리거
- 세션 종료 시
- 복잡한 작업 완료 후

## 반성 체크리스트

1. **이번 세션에서 잘한 것** — 어떤 접근이 효과적이었나?
2. **개선이 필요한 것** — 실패·비효율 부분?
3. **새로운 인사이트** — 사용자·패턴에서 발견한 것?

## 업데이트 액션
- MEMORY.md 학습된 패턴 테이블에 추가
- 개선된 절차는 해당 SKILL 파일 업데이트
- 새 자동화 필요시 스케줄러 생성
